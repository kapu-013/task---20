# LIVE DEMO SCRIPT

## Total Time: 5-7 minutes

---

### Step 1: Start Application (30 seconds)

**Say:** "Let me first start our application"

**Actions:**
1. Open terminal
2. Type: cd project-folder
3. Type: npm start
4. Show: "Server running on http://localhost:3000"

---

### Step 2: Open Browser (15 seconds)

**Say:** "Now I'll open the browser to show the application"

**Actions:**
1. Open Chrome/Firefox
2. Type: http://localhost:3000
3. Show homepage

---

### Step 3: Show Homepage (30 seconds)

**Say:** "This is our homepage. It shows..."

**Explain:**
- What users see first
- Navigation menu
- Main features

---

### Step 4: CREATE Operation (1 minute)

**Say:** "Let me demonstrate how to add data"

**Actions:**
1. Click 'Add New' button
2. Fill the form:
   - Name: "Test User"
   - Email: "test@example.com"
3. Click Submit
4. Show success message

---

### Step 5: READ Operation (1 minute)

**Say:** "Here's how we view data"

**Actions:**
1. Show list of all records
2. Click on any record
3. Show detailed view
4. Open browser console
5. Show API response (JSON)

---

### Step 6: UPDATE Operation (1 minute)

**Say:** "Now let me update existing data"

**Actions:**
1. Click Edit button on a record
2. Change name to "Updated Name"
3. Click Save/Update
4. Show updated record

---

### Step 7: DELETE Operation (1 minute)

**Say:** "Finally, let me delete a record"

**Actions:**
1. Click Delete button
2. Confirm deletion
3. Show record removed from list
4. Refresh page to confirm

---

### Step 8: Show Database (30 seconds)

**Say:** "Let me show how data is stored"

**Actions:**
1. Open MongoDB Compass / phpMyAdmin
2. Show the database
3. Show the collection/table
4. Point to the data we just created

---

### Step 9: API Testing (1 minute)

**Say:** "Let me test the API directly"

**Actions:**
1. Open Postman
2. Test GET request: /api/users
3. Test POST request with new data
4. Show JSON response

---

### Backup Plan (If demo fails)

**If server fails:**
- Show screenshots of working app
- Show recorded video
- Explain code instead

**Commands to fix quick:**
- 
pm start - restart server
- 
ode app.js - run directly
- 
etstat -ano | findstr :3000 - check port
