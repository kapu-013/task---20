🔌 REST APIs

Endpoints for data operations

POST /api/users - Create new user
PUT /api/users/:id - Update user

⚠️ Challenges & Solutions

Problems faced and how I solved them

1

Database Connection Failed

MongoDB connection refused error

✅ Solution: Started MongoDB service and verified connection string
2

CORS Policy Error

Browser blocked API requests from different origin

✅ Solution: Added CORS middleware in Express app.use(cors())
3

Empty Data Submission

Users submitting blank forms

✅ Solution: Implemented validation on both frontend and backend
4

Time Management

Features taking longer than estimated

✅ Solution: Prioritized MVP features, used agile methodology
